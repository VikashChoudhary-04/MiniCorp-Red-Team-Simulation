# Windows Client

- Role:

  - Corporate workstation inside the MiniCorp domain.

- Operating System:

  - Windows 10

---

## Network Configuration

- IP Address:

  - 192.168.56.30

- Domain:

  - minicorp.local

---

## Purpose in Lab

- This machine simulates a typical employee workstation.

- Attackers who gain internal access may attempt:

  - Credential harvesting
  - Lateral movement
  - Domain privilege escalation

---

## Example Attack Opportunities

- Examples of attacks against domain-joined workstations:

  - SMB credential capture
  - Pass-the-Hash attacks
  - Kerberos ticket abuse
  - Local privilege escalation
